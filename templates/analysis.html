models/analysis

The sentiment analysis of r/doge Reddit thread comments was processed and performed using VADER, a sentiment analysis tool attuned to social media. The extraction of comments for sentiment polarity comparison to stock price movement was performed using the total comments of each daily discussion thread of the r/doge subReddit within the relative time period to determine congruency with the cryptocurrency stock price for the duration of 3 months at 30-minute intervals. Extraction of the comments from each daily discussion thread was made using PRAW (Reddit API) for thread id extraction, and PMAW, a 3rd-party wrapper for pushshift.io that supports multi-threading and batch requests for comment extraction of each thread ID. A total of 554K comments were ran through VADER, a model trained on NYT OPEds, IMDB Reviews, Amazon Reviews, and Twitter Feeds. From there each comment was attributed with a Polarity/Compound Score which was determined to be of an overall positive, negative or neutral sentiment based on its -1 to 1 value being above 0.05 rated as Positive, below -0.05 rated as Negative, and between -0.05 and 0.05 rated as Neutral, for an overall comment score.


Naive Bayes Text Classifier and Prediction Accuracy based on VADER Sentiment Analysis Categorization

Based on the VADER categoration of each comment score, the dataset of comments as well as it's attributed score was then input into an ML pipeline to label and determine features. A Naive Bayes model was then created to fit the training data and predict a sentiment of Positive, Negative or Neutral for each comment in the testing data. The overall accuracy of the model at predicting reviews was 0.77% accurate.

The full NB process of classification can be viewed here.

Random Forests Text Classifier and Prediction Accuracy based on VADER Sentiment Analysis Categorization
The same data was taken and processed for comment features and vectorized to fit a TFIDF feature matrix. The data was then split for training and testing and ran through Random Forest Classifier. The results of the RFC predictions were then determined in a classification report and the overall accuracy of the model at predicting reviews was determined to be 0.86% accurate.
